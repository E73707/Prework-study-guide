Git

    Written below are some steps/notes on specific commands taking priority, other less common commands are present in the cheat sheet linked below it is important to note that any quotation marks in the passage below with * are to be replaced by the less/greater-than signs, double quotation marks indicate the use of quotation marks as well as the preceding symbols
    Cheat sheet
  • Git init *"repository name"* is used to start an empty repo or to initialise an existing one in the project root
  • Git commit -m *""commit message""* saves local changes to the branch you are locally working on
  • Git status: checks what branch we are currently on
  • Git checkout *"branch name" switches to a different currently existing branch
  • Git checkout -b *"branch-name"*: creates a new branch and switches to it
  • Git diff: shows the unstaged commands
  • Git pull: fetches the changes from the remote repo and merges any remote changes to the local branch
  • Git add: *"file path"* is used to stage changed files
  • Git branch: lists the branch names available for selection
  • Git push: pushes the committed changes to the remote branch
    • if the branch is not yet tracked and is only on the local machine run the command like the following
      • git push --set-upstream *"remote branch"* *"branch name"*
  • Git clone: downloads the latest version of a remote project and copies it to the selected location on the local machine
    • example: git clone *"repository of url"*
    • you can also clone a specific branch by adding in the following command
      • git clone *"repository url"* -b *"branch name"*
  • To save your changes to the main branch:
    • git add -A
      • adds a file as it looks now to your next commit (stage)
    • git commit -m "your message here"
      • commit your staged content as a new commit snapshot
    • git pull origin main
      • Pull in the base branch (main) to our feature branch
    • git push origin feature/______
      • Transmits local branch commits to the remote repository branch
